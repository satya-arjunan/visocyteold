Visocyte

Visocyte is an open-source, multi-platform data analysis and visualization application. Visocyte users can quickly build visualizations to analyze their data using qualitative and quantitative techniques. The data exploration can be done interactively in 3D or programmatically using Visocyte's batch processing capabilities.

Visocyte was developed to analyze extremely large datasets using distributed memory computing resources. It can be run on supercomputers to analyze datasets of terascale as well as on laptops for smaller data.

For the user's manual, refer to the Visocyte User's Manual, a pdf for which is included in your binary distribution or can be obtained from http://www.visocyte.org/download/.
